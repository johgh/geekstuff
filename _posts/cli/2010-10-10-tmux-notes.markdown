List commands
: `^A:lscm`

List current mappings
: `^A?`

Show clients/sessions/terminals
: `^A:info`

Choose split window
: `^A<arrow_keys>`

Change window size
: `^A<arrow_keys>` *(keep pressed)*

Alternate layout
: `^A<space>`

Break pane (fullscreen)
: `^A!`

# Custom commands
> see config [here](https://github.com/johgh/dotfiles/blob/master/.tmux.conf)

Fullscreen current window
: `^A|`

Go to last window
: `^A^A`
